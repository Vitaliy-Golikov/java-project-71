## Описание
Программа для сравнения двух конфигурационных файлов и вывода их различий.

## Требования
- Java 21
- Gradle (или использование обёртки `gradlew`)

## Установка и запуск
```bash
# Сборка проекта
make build

# Запуск программы
make run

# Запуск тестов
make test
